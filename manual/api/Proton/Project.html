Proton::Project

A project.

Common usage

Instanciating:

project = Project.new('~/spur')
project == Proton.project          # the last defined project

Building:

project.build
project.build  |file| puts "Building #file..." }

Getting pages:

Proton::Page['/index.html']        # ~/spur/index.md; uses Proton.project

Configuration:

project.config_file              # ~/spur/hyde.conf
project.config                   # Config from above file (OpenStruct)
project.config.site_path

Paths:

project.path(:site)              # ~/spur/site (based on config site_path)
project.path(:extensions)

project.root('a/b', 'c')         # ~/spur/a/b/c

Indexing:

project.pages                    # [<#Page>, <#Page>, ...]
project.files                    # ['/index.md', '/style.sass', ...]
                                 # (only site files)
project.ignored_files

Methods

  • build Builds.
  • config Returns a Proton::Config instance.
  • config_file Returns the configuration file for the project.
  • files Returns the site files for the project, free from the ignored files.
  • ignored_files Returns the files to be ignored for the project.
  • pages Returns the pages for the project.
  • path Returns the path for a certain aspect.
  • root Returns the root path of the project.
